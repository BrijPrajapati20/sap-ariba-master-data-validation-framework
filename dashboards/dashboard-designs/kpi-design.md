# Validation KPI Design

## Core Validation KPIs

| KPI | Description |
|---|---|
| Total Records | Total records processed |
| Passed Records | Successfully validated records |
| Failed Records | Validation failed records |
| Validation Accuracy % | Passed / Total |
| Duplicate Records | Duplicate entries identified |
| Missing Records | Missing entries in target system |
| Mismatch Records | Field mismatch count |
| Null Records | Mandatory field failures |

---

# Severity KPIs

| Severity | Description |
|---|---|
| Critical | Business-critical validation failures |
| Error | Standard validation failures |
| Warning | Informational discrepancies |
| Info | Audit information |

---

# Trend KPIs

| KPI | Description |
|---|---|
| Daily Failures | Daily validation failures |
| Weekly Trend | Weekly validation trend |
| Top Failing Master Data | Most problematic master data |
| Region-wise Quality | Country/Region quality score |
